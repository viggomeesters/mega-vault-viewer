# Mega Vault Viewer v0.1.0

Initial public MVP release of Mega Vault Viewer.

## Included

- Local-first macOS desktop app built with Tauri.
- Markdown reader with WikiLink navigation.
- Local SQLite metadata/link index.
- Local Tantivy full-text search.
- Mixed-format vault item viewing for Markdown, YAML, JSON, JSONL, images, PDFs, and generic files.
- Rebuildable runtime state outside the vault.
- Public-safe fixture/demo assets and repository documentation.

## Verification

Release preparation used:

```bash
cargo fmt --all -- --check
cargo test --workspace
cargo clippy --all-targets -- -D warnings
npm test --if-present
npm run build --if-present
npm run desktop:build
git diff --check
```

## Important Caveats

- The macOS app bundle is unsigned.
- The app is not notarized.
- There is no auto-update channel.
- This is an MVP release intended for local evaluation and development.
- Use fixture or non-private vault data when reporting public issues.
